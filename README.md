## 🧑‍⚖️ 그라운드 룰

- 주중(월~금) 1개 이상의 질문에 대한 답변을 작성합니다.
- 주말(토~일) 온라인 모의 인터뷰를 진행합니다.
- 답변의 양식은 자유롭게 스스로에게 도움이 되는 방식으로 작성합니다.
- 스스로 필요한 꼬리 질문을 1개 이상 생각해봅니다.(권장)
